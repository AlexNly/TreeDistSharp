# Attribution

TreeDistSharp is a C# port of the R [TreeDist](https://github.com/ms609/TreeDist) package.

## Original Authors

**TreeDist R Package**
- Martin R. Smith (maintainer)
- Repository: https://github.com/ms609/TreeDist
- License: GPL-3.0

**Linear Assignment Problem (LAP) Solver**
- Roy Jonker (original algorithm)
- Yong Yang (C++ implementation)
- Yi Cao (MATLAB implementation)
- Based on: Jonker & Volgenant (1987) "A shortest augmenting path algorithm for dense and sparse linear assignment problems"
